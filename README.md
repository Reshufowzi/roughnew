# Kubernetes Setup

## Install kubectl

```bash
curl -LO https://dl.k8s.io/release/v1.29.0/bin/linux/amd64/kubectl
chmod +x kubectl
sudo mv kubectl /usr/local/bin/
```

## Check kubectl version

```bash
kubectl version --client
```
